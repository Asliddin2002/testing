# TypeScript Calculator Demo

A simple **calculator implemented in TypeScript** with unit tests and a live demo UI.  
This project is designed for practicing TypeScript, testing, and building a minimal browser-based calculator.

---

## Features

- Simple calculator logic: `+`, `-`, `*`, `/`, decimal numbers
- Unit tests with Jest
- Live demo in the browser
- Fully written in TypeScript

---

## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Asliddin2002/testing.git
cd testing
```

### 2. Install dependencies

```bash
npm install
```

### 3. Compile TypeScript

```bash
npx tsc
```

### 4. Run the demo in the browser

```bash
npm install -g serve
serve .
```

### 5. Run unit tests

```bash
npm test
```
